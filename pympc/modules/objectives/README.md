This directort contains the objectives for the model predictive control optimization problem. 

Each Objective should impose soft constaints on the solver. Objectives should be based off of the mathematical formulations defined in 

https://github.com/tud-amr/mpc_planner

and https://github.com/oscardegroot/scenario_module

Objectives should all use a standardized interface.

Objectives should each have an appropriate visualization method which lets them be integrated into the final GIFs generated by testing.

Conbtouring constriants should always be used when the contouring objective is used. 

Each Objective should have logical settings configurable by the CONFIG.yml file. 

You should be able to use a single type of objective at once. 

All objectives should provide meaningful info and debug level logging for their operations.
